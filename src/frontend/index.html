Ayuda del sistema

¿Qué es SIGAT?

SIGAT (Sistema de Gestión de Atletas de Taekwondo) es el sistema de registro y seguimiento de los atletas de la Asociación de Taekwondo del Estado Miranda. Permite llevar la ficha completa de cada atleta (datos personales, escuela, entrenador, cinturón, periodos de actividad y representante), generar reportes en PDF y respaldar toda la información.

Desarrollado por el Ingeniero de Sistemas Ricardo Pacheco.

¿Cómo está hecho?

Es una aplicación ultra-portátil: un único ejecutable (app.exe) que no requiere instaladores, ni servidores externos, ni conexión a internet. Al abrirlo levanta un pequeño servidor local y abre el navegador en http://localhost:8080.

  • Backend: Go (un solo binario nativo, sin dependencias externas).
  • Base de datos: SQLite en el archivo app.db (se crea solo junto al ejecutable).
  • Interfaz: HTML, CSS y JavaScript embebidos dentro del propio ejecutable.
  • Tiempo real: WebSockets — los cambios de un usuario se reflejan al instante en los demás. El punto junto al título SIGAT indica la conexión en vivo (amarillo = conectado).

Roles y acceso

  • Administrador: acceso total — crear, editar y eliminar atletas, gestionar escuelas, entrenadores, usuarios, datos maestros y respaldos.
  • Consultor: solo lectura — puede buscar, consultar fichas y generar PDF, pero no modifica datos.

Cada usuario puede cambiar su nombre de usuario y contraseña desde Mi cuenta (arriba a la derecha).

Respaldo y restauración

Toda la información vive en el archivo app.db, ubicado en la misma carpeta que el ejecutable. Cuando se suban fotos o documentos de atletas, estos se guardarán en una carpeta data/ junto al ejecutable.

Para hacer un respaldo completo:

  1. Vaya a Respaldo → Descargar base de datos para obtener una copia de app.db.
  2. Copie también la carpeta data/ (si existe) para conservar fotos y documentos.
  3. Guarde ambos en un lugar seguro (pendrive, disco externo, nube).

Para restaurar en otra máquina o recuperar datos:

  1. Cierre el sistema (cierre la ventana del servidor).
  2. Reemplace el archivo app.db por su respaldo, y la carpeta data/ si la tiene.
  3. Vuelva a abrir app.exe: la información restaurada aparecerá tal cual.

También puede exportar o importar tablas individuales en formato CSV (por ejemplo, solo las escuelas). Ojo: importar un CSV reemplaza por completo el contenido de esa tabla.

Búsqueda y filtro avanzado

La barra de búsqueda encuentra por nombre, apellido o cédula. Los selectores de abajo filtran por escuela, municipio, cinturón y estado (activo/retirado). Para condiciones más precisas use Filtro avanzado, que combina varias condiciones tipo campo → operador → valor.

Coincidir «todas» exige que se cumplan todas las condiciones; «cualquiera» basta con que se cumpla una.

Ejemplos:

  • Atletas activos de cinturón Negro: coincidir todas → «Estado = activo» y «Cinturón = Negro».
  • Menores inscritos este año: coincidir todas → «Edad < 18» y «Fecha de inscripción ≥ 2026-01-01».
  • Atletas de la escuela A o la escuela B: coincidir cualquiera → «Escuela = A» y «Escuela = B».

Con los filtros aplicados puede generar un PDF (con filtros) que incluye solo los atletas que coinciden, más un resumen agregado.
